ValidatorList::load

Load configured trusted keys.

Synopsis 
bool
load(
    PublicKey const& localSigningKey,
    std::vector< std::string > const& configKeys,
    std::vector< std::string > const& publisherKeys);
Description
Parameters

Name

Description

localSigningKey

This node's validation public key

configKeys

List of trusted keys from config. Each entry consists of a base58 encoded validation public key, optionally followed by a comment.

publisherKeys

List of trusted publisher public keys. Each entry contains a base58 encoded account public key.

Thread Safety

May be called concurrently

Return Value

false if an entry is invalid or unparsable
