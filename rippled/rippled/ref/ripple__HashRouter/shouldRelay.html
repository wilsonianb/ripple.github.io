HashRouter::shouldRelay

Determines whether the hashed item should be relayed.

Synopsis 
boost::optional< std::set< PeerShortID > >
shouldRelay(
    uint256 const& key);
Description

Effects: If the item should be relayed, this function will not return true again until the hold time has expired. The internal set of peers will also be reset.

Return Value

A boost::optional set of peers which do not need to be relayed to. If the result is uninitialized, the item should not be relayed.
