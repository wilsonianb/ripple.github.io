FeeVote::Setup

Fee schedule to vote for.

Synopsis 
struct Setup
Data Members

Name

Description

account_reserve

The account reserve requirement in drops.

owner_reserve

The per-owned item reserve requirement in drops.

reference_fee

The cost of a reference transaction in drops.

reference_fee_units

The cost of a reference transaction in fee units.

Description

During voting ledgers, the FeeVote logic will try to move towards these values when injecting fee-setting transactions. A default-constructed FeeVote::Setup contains recommended values.
