LedgerEntryType

Ledger entry types. 

enum LedgerEntryType

Values

ltANY

Special type, anything This is used when the type in the Keylet is unknown, such as when building metadata.

ltCHILD

Special type, anything not a directory This is used when the type in the Keylet is unknown, such as when iterating.

ltINVALID
ltACCOUNT_ROOT
ltDIR_NODE

Directory node.

ltRIPPLE_STATE
ltTICKET
ltSIGNER_LIST
ltOFFER
ltLEDGER_HASHES
ltAMENDMENTS
ltFEE_SETTINGS
ltSUSPAY
ltPAYCHAN
ltNICKNAME
ltNotUsed01

These are stored in serialized data.

Remarks

Changing these values results in a hard fork.
