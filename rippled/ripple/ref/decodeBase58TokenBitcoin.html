decodeBase58TokenBitcoin

Decode a Base58 token using Bitcoin alphabet. 

std::string
decodeBase58TokenBitcoin(
    std::string const& s,
    int type);

The type and checksum must match or an empty string is returned.

This is used to detect user error. Specifically, when an AccountID is specified using the wrong base58 alphabet, so that a better error message may be returned.
