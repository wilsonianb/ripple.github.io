sign

Sign an STObject.

Synopsis 
void
sign(
    STObject& st,
    HashPrefix const& prefix,
    KeyType type,
    SecretKey const& sk,
    SF_Blob const& sigField = sfSignature);
Description
Parameters

Name

Description

st

Object to sign

prefix

Prefix to insert before serialized object when hashing

type

Signing key type used to derive public key

sk

Signing secret key

sigField

Field in which to store the signature on the object. If not specified the value defaults to sfSignature.

Remarks

If a signature already exists, it is overwritten.

Header

#include <ripple/protocol/Sign.h>
