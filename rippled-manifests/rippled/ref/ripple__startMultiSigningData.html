startMultiSigningData

Break the multi-signing hash computation into 2 parts for optimization.

Synopsis 
Serializer
startMultiSigningData(
    STObject const& obj);
Description

We can optimize verifying multiple multisignatures by splitting the data building into two parts; o A large part that is shared by all of the computations. o A small part that is unique to each signer in the multisignature.

The following methods support that optimization:* startMultiSigningData provides the large part which can be shared. * finishMuiltiSigningData caps the passed in serializer with each signer's unique data.
